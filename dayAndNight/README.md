# Day and Night - 双主题页面集合

这是一个支持白天模式和暗黑模式（Dark Mode）的页面集合，包含支付成功页面和红包封面选择页面。

## 功能特性

- ✅ **双主题支持**：白天模式和暗黑模式
- ✅ **自动主题检测**：根据系统设置自动切换主题
- ✅ **手动主题切换**：
  - 右下角主题切换按钮
  - **键盘快捷键**：Tab 键切换主题
- ✅ **主题持久化**：用户选择的主题会保存到 localStorage
- ✅ **流畅动画**：主题切换和成功图标动画
- ✅ **响应式设计**：适配移动端和桌面端
- ✅ **智能键盘交互**：Tab 键只在非交互元素上触发主题切换

## 使用方法

### 1. 直接打开
在浏览器中打开对应的 HTML 文件即可查看：
- `success.html` - 支付成功页面
- `redpacket.html` - 红包封面选择页面

### 2. 本地服务器（推荐）
```bash
# 使用 Python
python3 -m http.server 8000

# 使用 Node.js (http-server)
npx http-server -p 8000

# 然后访问
http://localhost:8000/dayAndNight/success.html
http://localhost:8000/dayAndNight/redpacket.html
```

### 3. 主题切换
- **自动切换**：页面会根据系统设置自动应用对应主题
- **手动切换**：
  - 点击右下角的主题切换按钮
  - 按下 **Tab 键**（当焦点不在输入框或按钮上时）
- **持久化**：用户手动选择的主题会保存，下次打开时自动应用

## 技术实现

### CSS 变量系统
使用 CSS 变量（Custom Properties）实现主题切换：

```css
:root {
  --bg-primary: #FFFFFF;
  --text-primary: #000000;
  /* ... */
}

[data-theme="dark"] {
  --bg-primary: #000000;
  --text-primary: #FFFFFF;
  /* ... */
}
```

### JavaScript 主题管理
- `ThemeManager` 类管理主题状态
- 自动检测系统主题偏好
- 支持手动切换和持久化存储

### 动画效果
- 成功图标：缩放动画（scale animation）
- 主题切换：平滑过渡（0.3s ease）
- 按钮交互：点击反馈动画

## 自定义配置

### 修改颜色
在对应的 CSS 文件中修改 CSS 变量（两个页面使用相同的色彩体系）：

```css
/* success.css 或 redpacket.css */
:root {
  --success-color: #07C160;  /* 修改成功色 */
  --button-bg: #F5F5F5;      /* 修改按钮背景 */
  --tab-active-color: #FF3B30; /* 标签页激活色 */
  /* ... */
}

[data-theme="dark"] {
  --success-color: #1AAD19;  /* 暗黑模式成功色 */
  --tab-active-color: #FF453A; /* 暗黑模式标签激活色 */
  /* ... */
}
```

### 修改尺寸
调整容器尺寸：

```css
.app-container {
  width: 375px;   /* 修改宽度 */
  height: 812px;  /* 修改高度 */
}
```

## 红包页面交互功能

### 标签页切换
- 点击标签按钮切换内容
- 当前标签显示红色下划线
- 标签文字颜色变化（选中：主要文本色，未选中：次要文本色）

### 红包封面选择
- 点击任意卡片进行选择
- 选中的卡片显示绿色边框和阴影效果
- 卡片名称变为绿色并加粗
- 只有选中卡片后，"使用"按钮才可点击

### 使用按钮
- 未选中时：禁用状态（半透明，不可点击）
- 选中后：激活状态（可点击，有悬浮效果）
- 点击后：显示选中卡片的 ID（实际应用中可导航到下一步）
